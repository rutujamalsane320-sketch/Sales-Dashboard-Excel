Sales Dashboard Project

Project Overview
This project is an interactive Sales Dashboard created in Microsoft Excel to analyze sales performance and business trends.

Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

Key Features
- Total Sales Analysis
- Monthly Sales Trends
- Product-wise Sales Performance
- Category-wise Sales Distribution
- KPI Metrics Dashboard
- Interactive Filters

Insights
- Identified top-performing products
- Analyzed monthly sales growth
- Compared category performance
- Tracked overall business performance
